# React TS Vite Test

Тестовое задание на `React + TypeScript + Vite`: страница карточки автомобиля с галереей, характеристиками, комплектацией, секцией похожих автомобилей и фиксированными кнопками действий.

## Стек

- React 19
- TypeScript
- Vite
- Swiper
- ESLint

## Запуск

```bash
npm install
npm run dev
```

## Полезные команды

```bash
npm run build
npm run lint
npm run preview
```

## Структура

- `src/features/product-page` — основная feature страницы
- `src/assets/fonts` — шрифты
- `src/assets/images/product-page` — изображения галереи, баннера и похожих авто
- `public/favicon.svg` — favicon проекта
